Dossiers personnalisables
Crée autant de dossiers que tu veux, avec un nom et une couleur. Les onglets Google Sheets se colorient automatiquement.
Contrôle de visibilité
Masque ou affiche un dossier entier d'un clic, ou feuille par feuille.
Navigation directe
Clique sur n'importe quelle feuille dans la sidebar pour y aller directement. Plus besoin de faire défiler 40 onglets.
Suppression flexible
Supprime un dossier en choisissant : dissocier les feuilles (les garder) ou les supprimer définitivement avec lui.

Installation

En place en 2 minutes.

Aucune installation système. Tout se passe dans l'éditeur Apps Script intégré à Google Sheets.

1
Ouvre l'éditeur Apps Script
Dans ton Google Sheet, clique sur Extensions → Apps Script. L'éditeur s'ouvre dans un nouvel onglet.
2
Colle le fichier Code.gs
Remplace le contenu du fichier Code.gs par le code du dépôt GitHub.
3
Crée le fichier Sidebar.html
Clique sur + → HTML → nomme-le exactement Sidebar → colle le contenu.
4
Sauvegarde et rafraîchis
Ctrl+S dans l'éditeur, puis rafraîchis ton Google Sheet. Le menu 📁 Dossiers apparaît dans la barre de menu.
